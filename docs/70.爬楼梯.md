力扣链接：<a href="https://leetcode.cn/problems/climbing-stairs/description/" target="_blank">70.爬楼梯</a>

难度：⭐ <br/>

解题关键词：动态规划、斐波那契<br />

解题思路：到达第 n 个台阶，其实等于 到达第 n-1 个台阶的方法 + 到达第 n-2 个台阶的方法 总和。<br />

```typescript
function climbStairs(n: number): number {
  // p代表 n - 2
  // q代表 n - 1
  // r代表 p + q
  let p = 0,
    q = 0,
    r = 1;

  // 当i到了n 那么返回 r
  for (let i = 1; i <= n; i++) {
    p = q;
    q = r;
    r = p + q;
  }

  return r;
}
```
